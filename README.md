# STARTVOLT page scaffold

Локальная заготовка страницы по базовым стилям startvolt.com.

## Файлы

- `academy.html` - основная страница Академии STARTVOLT.
- `assets/css/startvolt-base.css` - шрифты, переменные, базовые стили, хедер, футер и контентные заготовки.
- `assets/fonts/` - локальные Franklin-шрифты, выгруженные с сайта.
- `assets/img/` - логотип STARTVOLT, логотип TecDoc и hero-изображение.
- `assets/js/startvolt-navigation-menu.js` - скрипт хедера и мобильного меню.

Открывать можно напрямую через `academy.html` или через любой локальный сервер.
